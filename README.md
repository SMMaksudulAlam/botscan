**Artifact Overview**

This repository contains the code and artifacts used in our experiments for identifying and probing C2 servers using active malware execution and replay-based probing. Due to platform limitations and security considerations, some datasets and artifacts are partially shared or summarized.

**Directory Structure**

#scanner_tool/
Contains the implementation of the scanner used in active probing.
-- scaled_replay: given a segment, probe all the ips in that segment
-- map_probe: loads the payload data and prepares it for use by the scanner

#malware_network_traces_extract_analysis/
--Contains scripts and notebooks for extracting and analyzing network traces generated by malware execution.

#thought_experiment/
-- artifacts related to Evaluation 2.
-- validation_2.ipynb: notebook used to reproduce the corresponding experiment and analysis

#behavioral_profiling
-- Collate data from different sources and build BP-DS
-- Analsyze dataset for generating insights

#behavior_adaptive_algorithm
-- behabior adaptive algorithm implementation: the scanner module is kept in a different directory for organization purpose
-- the datapoint of active scanning in the wild 

#pcap_samples/
-- example PCAP files generated by active malware execution. Due to data size and platform upload limitations, only a subset of the traces is shared.

#resource_consumption_measurement/
-- data and scripts used to compare resource consumption between malware-activation-based probing and replay-based probing.

#result_verification_with_VT/
-- code used to verify experimental results using the VirusTotal API.

#spatial_experiment_result_snapshot/
-- data and intermediate results for experiments analyzing spatial characteristics of C2 infrastructure.

#temporal_experiment_result_snapshot/
-- data and intermediate results for experiments analyzing temporal characteristics of C2 infrastructure.

#Experimental_Results.xlsx
-- a summary of experimental results reported in the paper.



**Notes on Data and Malware Availability**
For security and safety reasons, malware binaries used in our experiments are not shared as part of this artifact. Instead, we provide network traces, extracted features, and analysis results that are sufficient to reproduce and validate our findings without distributing live malware.

Additionally, some large datasets, such as full PCAP traces generated during malware execution, cannot be fully shared in this repository due to platform size limitations. Where applicable, representative samples and processed results are provided.
